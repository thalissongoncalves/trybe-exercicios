# UNIX, SHELL, GIT E GITHUB

Esse bloco pertence ao módulo de `fundamentos` do curso da Trybe

Nessa seção aprendi a utilizar o terminal do sistema operacional através do `Unix & Shell`, explorando comandos para navegar entre diretórios, criar, mover e editar arquivos, além de automatizar tarefas.

Em seguida, aprendi a trabalhar com o sistema de controle de versão `Git`, entendendo como registrar alterações, criar ramificações (branches) e gerenciar o histórico de um projeto. Também aprendi a utilizar o `GitHub` para armazenar e compartilhar repositórios remotamente, colaborando de forma organizada com outros desenvolvedores.

Com isso, me tornei capaz de utilizar o terminal para realizar operações no sistema, versionar códigos com segurança e colaborar em projetos por meio de ferramentas amplamente utilizadas no mercado.

Tendo isso em mente, esta pasta contém todas as atividades de aprendizagem que desenvolvi ao longo da seção de `Unix, Shell e Git` do curso da [Trybe](https://www.betrybe.com/).
